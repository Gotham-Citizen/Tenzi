# Tenzies

A simple dice game built using React, CSS and JavaScript.

## Game rules
First look at the 10 dice that has been presented on the page. Choose a number that appears the most. Then click the dice with that number to hold it. Then hit roll again to find the new dice to hold that has the number you’re going for. Continue this process until all 10 dices hold the same number then you win the game.
